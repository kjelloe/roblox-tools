8. Multiplayer effects in the CutsceneServer path — DONE (2026-07-14, client-fired)

optional   decision only — matters only for real multiplayer servers

In the synchronized CutsceneServer path, effect bursts fire server-side; ParticleEmitter:Emit() does not replicate, so clients in a live server may not see them (fine in Studio play-solo; subtitles and camera are already client-side). If you plan multiplayer cutscenes with particle effects, ask Claude to move effect firing client-side in that path; otherwise ignore — the documented caveat covers it.
